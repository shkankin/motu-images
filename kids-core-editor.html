How this maps to the app
These figures live in the browser's localStorage under the key motu-kids-core. The MOTU Vault app reads them on startup and merges them into the Kids Core line on top of any figures shipped via the repo's kids-core.json.

If you want a figure visible to everyone using the app, commit it to kids-core.json in the motu-images repo. If you want a figure visible only to you, leave it here in localStorage.

Important: localStorage is per-origin. This editor only sees the motu-kids-core list if you open it from the same domain as the deployed app. If you open it via file://, you can still build a list and export the JSON to paste into the deployed app (use Settings → Import or paste into localStorage.setItem('motu-kids-core', ...) via the browser console).
